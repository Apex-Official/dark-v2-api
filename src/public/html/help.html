Pinterest Search Example

Endpoint:

https://dark-api-x.vercel.app/api/v1/search/pinterest?query=anime

Response:

{
  status: true,
  creator: 'Anas radio',
  pins: [...]

Usage (fetch):

Notes:

  • Make sure the endpoint URL is correct.
  • Check your internet connection if the request fails.
  • Some endpoints may be temporarily down.
  • Ensure your query parameters are properly URL encoded.

Do I need an API key?
Most endpoints require a valid API key. Make sure you include it in your request headers.
How can I use the API?
Visit the home page and choose what suits you from available APIs.
What if I get a 500 error?
Retry after a few seconds. If it persists, contact support.
